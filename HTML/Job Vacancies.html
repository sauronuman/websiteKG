Job Specifications
On this page, you will find information on job specifications as a web developer as well as how to kick-start your application if you want to apply to work for us. Here is a list of requirements and responsibilities as a web developer.
Web Developer Requirments
Here is a list of requirements that a web developer must be able to fulfill to be successful.
   
- Bachelor's degree in web development or any related field or relevant experience in web development.
-Adequate knowledge and previous experience in programming
-Competent in using HTML,CCS, Java Script and MySQL
-Can be a dedicated team player
Web Developer Responsibilities 
As with any job, there are some responsibilities involve. Here is a list of a few.
- Designing, building, testing and possibly maintaining a website or software application
-Working with teams to resolve any conflicts, develop content criteria, prioritize requirments and find and agree on solutions
-Performing and directing any website updates
-Writing, describing and editing website content and (OR) directing otherteam members that are producing content for a webiste
-Backing up files from websites to local directories so that they can be recovered
-Identify errors through testing or customer feedback and rectifying any errors or referring them to revevant employees to be rectified.
-Making sure that code meets industry standards and is compatible with a range of web browsers, operating systems and devices.
